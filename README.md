# Estifanos Beyene — Portfolio

A static developer portfolio built with SvelteKit, Tailwind CSS v4, and DaisyUI. Deployed to GitHub Pages.

## Tech Stack

- [SvelteKit](https://svelte.dev/docs/kit) with `@sveltejs/adapter-static`
- [Tailwind CSS v4](https://tailwindcss.com) + [DaisyUI v5](https://daisyui.com)
- TypeScript

## Pages

- `/` — Home / whoami
- `/projects` — Things I've built
- `/skills` — Tech I work with
- `/experience` — Professional journey
- `/about` — About me
- `/contact` — Get in touch

## Developing

```sh
npm install
npm run dev
```

## Building

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```

## Deployment

Deployed automatically to GitHub Pages via GitHub Actions on every push to `main`.

Live site: https://justEstif.github.io/portfolio
